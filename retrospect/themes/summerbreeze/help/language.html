The Language pull-down menu, when present, is found to the right of the Top Menu above each page. Clicking on the down arrow will display a list of languages currently available for Retrospect-GDS. The list includes:
English
Spanish
German
Dutch
French
Polish
Portugese
Norwegian
Hebrew
Swedish

Selecting one of these languages will change the tabs, menus, titles and basic information into the selected language. It will not however, change the notes, sources or data. The information in these areas will remain in the language they were entered in.

If the Language pull-down menu is not present, the website owner has elected not to offer this option.
