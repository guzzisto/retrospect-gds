The Search Page can be used to locate specific information based on Name, Location or Note.

Name Search
To search for a specific given and/or surname, enter the name to be searched for in the appropriate box. If you would like the search to be based on Soundex, check the box next to Soundex, and then click the Search button in the Name Search area. The Results page will then be displayed. If there are any matches the names will be listed. If there are no matches, the results page will display :

No matching records were found.

You may return to the Search Page by clicking on the Search Tab next to the Results Tab.

Location Search
To search for a location, enter the location into the Keywords box in the Location Search area. You may refine the matching using the Match Keywords pull-down menu. The options in this pull- down include: All, Any, Phrase, Starts With, Ends With. Once you have made your selections, click the Search button in the Location search area. The Results page will then be displayed. If there are any matches the names of persons with matches in their locations will be listed. If there are no matches, the results page will display :

No matching records were found.

You may return to the Search Page by clicking on the Search Tab next to the Results Tab.

Note Search
To search for keywords within the notes of the family history enter the keyword(s) in the Keywords box in the Notes Search area. You may refine the matching using the Match Keywords pull-down menu. The options in this pull-down include: All, Any, Phrase, Starts With, Ends With. Once you have made your selections, click the Search button in the Note search area. The Results page will then be displayed. If there are any matches the names of persons with matches in their Notes section will be listed. If there are no matches, the results page will display :

No matching records were found.

You may return to the Search page by clicking on the Search tab next to the Results tab.
